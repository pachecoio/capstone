# client

## Project setup

Create a .env file with the following environment variables

```
VUE_APP_DOMAIN=<YOUR AUTH0 DOMAIN CONFIG>
VUE_APP_CLIENT_ID=<YOUR AUTH0 CLIENT ID>
VUE_APP_AUDIENCE=<YOUR AUTH0 APP AUDIENCE>
```

```
yarn install
```
